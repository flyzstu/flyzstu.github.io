docker安装jenkins

发表于 2022-04-06

关键词：docker，jenkins

为什么要用docker？这个不在赘述。

最近要求部署的项目要求是内外网隔离的，代码跑起来估计要安装各种依赖，想想就头大。。。

现在考虑的是可以通过部署docker私有仓库，拉取镜像，跑容器。

为啥还需要jenkins呢，因为项目还没有开发完毕，但是交付的日期还有不到10天了。。。

安装

官方教程：https://www.jenkins.io/doc/book/installing/linux/

建议参考英文，看不懂的可以用谷歌翻译。

官网给出方法的是通过docker:dind镜像运行另外一个部署了docker daemon的容器（区别于宿主机），然后通过docker-workflow方式使用，这样就实现了宿主机和构建环境的隔离，但是这样有点麻烦。。。于是就采用了之前文章中说过的docker in docker 的用法。
