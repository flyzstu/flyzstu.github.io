发表于 2022-11-09

什么是DevOps？

DevOps的哲学暂且不表，使用Docker搭建nexus3私有镜像仓库。

关键词：nexus3

前言

在搭建K8s容器平台时，有些镜像需要经常性的被拉取，有时候外网带宽会被大量占用，这在业务上线后是不可取的，为了方便镜像的拉取，私有仓库的概念应运而生。

本文主要介绍使用nexus3搭建私有仓库。

docker
