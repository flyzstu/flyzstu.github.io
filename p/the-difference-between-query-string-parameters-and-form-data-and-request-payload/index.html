    http请求参数之Query String Parameters、Form Data、Request Payload区别

    发表于 2023-01-05

    在与server端进行数据传递时，通常会用到GET、POST方法进行参数提交，而参数提交的方式，通常取决于server端对数据的接收方式。

    关键词：http

    Query String Parameters

    Query String Parameters 当发起一次HTTP请求时，参数会以url string的形式进行传递。

    即?后面的字符串则为请求的参数，并以&为分隔符。

    header

    传入参数

    Request Payload

    当发起一次POST请求后，若content-type为application/json，则参数会以Request Payload的形式进行传递，数据形式为JSON。

    上述请求不会显式的出现在URL中。

    header

    传入参数

    Form Data

    当发起一次HTTP请求时，若未指定content-type，则默认content-type为application/x-www-form-urlencoded。

    参数会以From Data的形式进行传递，不会显式出现在请求url中。
